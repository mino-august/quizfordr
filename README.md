# 🎮 우아한퀴즈타임

카훗 스타일의 실시간 퀴즈 플랫폼

## 기능

- 🎯 실시간 멀티플레이어 퀴즈
- 📱 QR코드로 쉽게 참가
- 🏆 실시간 점수 및 랭킹
- 🎨 레트로 픽셀아트 디자인

## 문제 유형

- ✅ 객관식 (4지선다)
- ⭕ OX 퀴즈
- ✍️ 주관식
- 🎚️ 숫자 슬라이더

## 시작하기

```bash
npm install
npm start
```

## 기술 스택

- Node.js + Express
- Socket.io (실시간 통신)
- Vanilla JavaScript
